# Images Needed

Add the following images to this folder:

## Required Images

1. **legion-emblem.png** (80x80px or larger)
   - American Legion emblem
   - Download from https://www.legion.org/publications/emblem
   - Or use a transparent PNG version

2. **hero-1.jpg** (1920x600px recommended)
   - Main hero slider image
   - Suggest: Veterans or military-related image

3. **hero-2.jpg** (1920x600px recommended)
   - Second slider image
   - Suggest: Community event photo

4. **hero-3.jpg** (1920x600px recommended)
   - Third slider image
   - Suggest: Hall or facility photo

## Optional Images

- Post photos for gallery
- Event photos
- Officer photos
- Historical photos

## Image Guidelines

- Use high-quality, royalty-free images
- Optimize images for web (compress without losing quality)
- JPG format for photos, PNG for graphics with transparency
- Recommended tools: TinyPNG, ImageOptim, Squoosh

## Free Image Sources

- Unsplash (https://unsplash.com) - Search "veterans" or "military"
- Pexels (https://pexels.com)
- Department of Defense imagery (https://www.defense.gov/Multimedia/)
